排序演算法報告：Index 的應用

本報告展示兩種使用「Index (索引)」觀念來優化的排序演算法。

1. 進階氣泡排序法 (Advanced Bubble Sort)

Index 用法：紀錄每一輪最後一次發生交換的位置（Index）。該 Index 之後的元素皆已排序完成，下一輪只需比對到該位置即可，大幅減少比對次數。

// C 語言實作：進階氣泡排序
void advancedBubbleSort(int arr[], int n) {
    int n_temp = n;
    while (n_temp > 1) {
        int newn = 0; // 用來記錄最後一次交換的 index
        for (int i = 1; i < n_temp; i++) {
            if (arr[i - 1] > arr[i]) {
                // 交換數值
                int temp = arr[i - 1];
                arr[i - 1] = arr[i];
                arr[i] = temp;
                newn = i; // 更新最後交換的 index
            }
        }
        n_temp = newn; // 下一輪只需檢查到最後交換的位置
    }
}

2. 選擇排序法 (Selection Sort)

Index 用法：每次從未排序的區間中，找出最小值的「Index」，然後將該 Index 的數字與未排序區間的第一個數字交換。

// C 語言實作：選擇排序
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i; // 假設當前 i 是最小值的 index
        
        // 找出剩下的數字中最小的 index
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j; 
            }
        }
        // 如果最小值的 index 不是原來的 i，就進行交換
        if (minIndex != i) {
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }
}
